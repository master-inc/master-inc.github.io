About Bookzen

Vietnamese

Bookzen là một project nhằm giúp những người yêu thích sách tìm được nơi mua sách giá tốt nhất online.

Bookzen được viết hoàn toàn bởi Python với Scrapy để crawl dữ liệu sách và Flask làm framework chính cho website.

Giao diện(front-end) được thiết kế một cách đơn giản nhất và nhẹ nhất nhằm mang lại trải nghiệm tối ưu dành cho người sử dụng. Thay vì Bootstrap thì Bookzen sử dụng Bulma.io, một framework mới dựa trên Flexbox của Mozilla.

Bookzen sử dụng MongoDB làm database chính

Danh sách các địa chỉ mua bán sách online hiện đã có trong database của Bookzen:

  • Tiki
  • Lazada
  • Vinabook

Hiện Bookzen vẫn đang tiếp tục cập nhập cơ sở dữ liệu, sẽ được thông báo chi tiết hơn về kỹ thuật cũng như tiến độ trên blog của mình.

Mọi đóng góp về tính năng cũng như nâng cấp về source code có thể tham gia tại Github của Bookzen.

Để liên hệ, xin vui lòng truy cập /contact

English

Bookzen is a project that help "book worm" find where to buy favorite books with best price.

Bookzen is writen in Python, with Scrapy for data-crawling and Flask for website framework.

Front-end is using Bulma.io instead of Bootstrap for speed and light-weight experience for users. Bulma use Flexbox of Mozilla as main grid system, without Javascript in it.

MongoDB is using as main database for books

List site are already crawled for book data:

  • Tiki
  • Lazada
  • Vinabook

... and still have more. Bookzen will update it process and schedule on /blog.

All contribute of source code, please go to Bookzen's github page.

For contact, please go to /contact.
